## Design Context

### Users

Licensed Canadian therapists (psychotherapists, psychologists, counsellors) who regularly refer clients out. They need a structured, professional way to match clients with the right therapist by specialty, location, and modality. They use this tool during or between sessions — context is clinical, not casual.

### Brand Personality

**Three words:** Calm, clinical, structured

**Voice & tone:** Peer-to-peer. One therapist speaking to another. Professional but not corporate. Clear, not salesy. The interface should feel like a well-organized clinic — everything in its place, nothing unnecessary.

**Emotional goals:**

- **Relief and ease** — Finally, a simple tool that handles the messy referral process
- **Confidence and trust** — They feel safe referring clients through this system
- **Connection and community** — Part of a supportive professional network

### Aesthetic Direction

**Visual tone:** Clean, calm, structured. Warm neutrals with a teal accent. Think Psychology Today but better — less cluttered, more modern, properly organized.

**Reference:** Psychology Today (similar purpose: therapist directory) — but cleaner, more minimal, and less visually noisy.

**Anti-reference:** Anything flashy, gamified, or startup-feeling. No gradients, no bold illustrations, no playful animations. This is a clinical tool, not a consumer app.

**Theme:** Light mode default with full dark mode support. Warm cream backgrounds (#f7f4ef), not cold whites. Teal (#2a7c7c) as the singular brand accent — used sparingly for CTAs and interactive elements.

### Design Principles

1. **Structure over decoration** — Every element serves a purpose. No ornamental design. Use whitespace, hierarchy, and alignment to create clarity.

2. **Warm professionalism** — The palette is warm (cream, brown tones, teal) but the layout is clinical. This balance makes the tool feel approachable without being casual.

3. **One accent, used consistently** — Teal (#2a7c7c) is the only brand color. It marks interactive elements and primary actions. Everything else is neutral. This creates instant visual clarity about what's clickable.

4. **Content density over scroll depth** — Therapists are busy. Pack information efficiently. Use grids, tables, and compact layouts rather than spacious single-column marketing patterns.

5. **System-first, not page-first** — Design tokens and components, not one-off pages. Every color, spacing value, and radius comes from the token system in globals.css. No hardcoded values.

### Design Tokens Reference

- **Font:** Plus Jakarta Sans (400/500/600/700)
- **Brand:** --brand: #2a7c7c (teal), --brand-h: #1f6363, --brand-on: #fff
- **Backgrounds:** --bg: #f7f4ef (cream), --s1: #fdfbf8, --s2: #ffffff
- **Text:** --fg: #3d3836, --fg-2: #6b5e57, --fg-3: #776b63, --fg-4: #b5a99e
- **Spacing:** 4px base scale (--space-1 through --space-24)
- **Radii:** --radius-sm: 6px, --radius-md: 10px, --radius-lg: 14px, --radius-xl: 20px
- **Shadows:** --shadow-1 (subtle), --shadow-2 (elevated)
- **Styling pattern:** Inline style={{}} for CSS vars + Tailwind classes for layout
- **Accessibility:** WCAG AA compliance
